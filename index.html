What is SBTCVM?

SBTCVM is a project to develop a usable balanced ternary virtual machine, as well as develop the related tools, apis, programming languages, libraries and so on.

Projects:

SBTCVM Mark 2 is the latest and greatest VM developed by the SBTCVM project. It includes a variety of programs, and several utilities, such as a command shell, and a custom assembler.

Online Docs: https://sbtcvm.github.io/SBTCVM-Mark-2/
Repository: https://github.com/SBTCVM/SBTCVM-Mark-2


libbaltcalc is SBTCVM's balanced ternary integer mathematics backend and provides powerful balanced ternary integer support to python, It features both a lower level functional API and a high-level class-based API as well.

Online Docs: https://sbtcvm.github.io/libbaltcalc/
Repository: https://github.com/SBTCVM/libbaltcalc
